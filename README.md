# BM Program

## Getting started

#### Requirement

This app uses node v15.4.0, or npm v15.4.0, and yarn v1.16.0

#### Project setup

Install project dependencies you can use yarn:

```
yarn install
```

#### Run development server

```
yarn start
```

#### Run linters

```
yarn run lint
```

#### Run tests

```
yarn test
```

#### Create production build

```
yarn run build
```
